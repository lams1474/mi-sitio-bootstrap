# Semana 4 – CSS3 y Bootstrap 5

Proyecto desarrollado para la asignatura **Desarrollo de Aplicaciones Web** (Semana 4).  
El objetivo es crear una página web moderna, responsiva y publicada en GitHub Pages utilizando **CSS3 avanzado** y **Bootstrap 5**.

## Características

- Uso de **Bootstrap 5** mediante CDN (navbar, grid, cards, botones, collapse, formularios).
- Diseño **responsivo** usando el sistema de columnas (`row`, `col-md-4`) y clases de Bootstrap.
- Formulario con campos de **Nombre, Email y Comentarios**, usando `form-control` y contenedores responsivos.
- Estilos personalizados en `styles.css`:
  - Cambio de color de `.btn-primary`.
  - Sombras, bordes redondeados y efectos `:hover` con `transition` y `transform`.
  - `@media` propia para ajustar colores en pantallas pequeñas.
- Proyecto publicado con **GitHub Pages**.
